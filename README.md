# Mini-Story-Platform
Build a small writing and sharing tool where people can write stories, publish them, and others can read, like, and comment.
Try these credentials on account: amara@example.com / password123
How to start this? 
First step to start backened (terminal commands): cd inkwell\backend
copy .env.example .env
npm install
npm start
Second step to start frontend (terminal commands): cd inkwell\frontend
npm install
npm run dev
Then open http://localhost:5173 in your browser.
